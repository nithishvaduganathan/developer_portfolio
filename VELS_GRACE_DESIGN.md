## Responsive Design

### Mobile (< 768px)
- Single column product grid
- Stacked navigation
- Full-width buttons
- Simplified spacing
- Touch-friendly tap targets (minimum 44px)

### Tablet (768px - 1024px)
- Two column product grid
- Compact navigation
- Balanced layout

### Desktop (> 1024px)
- Three column product grid
- Full navigation with all items visible
- Optimal spacing and whitespace
- Hover effects fully utilized

## Typography

### Font Family
- Primary: `system-ui, -apple-system, sans-serif`
- Fallback to native system fonts for performance

### Font Sizes
- Hero Title: 3rem (2rem mobile)
- Page Titles: 2rem
- Section Headers: 1.5rem
- Product Names: 1.25rem
- Prices: 1.5rem (bold)
- Body Text: 1rem
- Small Text: 0.85rem

### Font Weights
- Titles: 700 (Bold)
- Headers: 600 (Semi-bold)
- Body: 400 (Regular)

## Animations & Transitions

### Hover Effects
- Cards: Lift 4px + shadow increase (0.3s ease)
- Buttons: Darken + lift 2px (0.3s ease)
- Links: Color change (0.3s ease)

### Click Effects
- Buttons: Scale 0.98 on active
- Quantity buttons: Scale 1.1 on hover

### Page Transitions
- Smooth scrolling
- Fade-in for modals (0.2s ease)

## Accessibility

### Color Contrast
- Brown on pastel green: WCAG AA compliant
- White on brown buttons: WCAG AAA compliant

### Interactive Elements
- Minimum 44x44px touch targets
- Clear focus states
- Semantic HTML structure
- Alt text for all images

### Forms
- Clear labels
- Error messages in red
- Required field indicators (*)
- Helpful placeholder text

## Special Effects

### Header Overlay
```css
background: rgba(255, 255, 255, 0.5);
backdrop-filter: blur(10px);
```

### Card Shadows
```css
box-shadow: 0 4px 12px rgba(93, 64, 55, 0.1);
/* Hover */
box-shadow: 0 8px 24px rgba(93, 64, 55, 0.2);
```

### Gradient Background
```css
background: linear-gradient(135deg, #e8f5e9 0%, #c8e6c9 100%);
```

This creates a subtle diagonal gradient from top-left to bottom-right, giving depth to the pastel green background.

## User Experience Notes

### Visual Hierarchy
1. Hero title and description (largest, centered)
2. Category filters (prominent, centered)
3. Product grid (balanced, equal emphasis)
4. Footer (minimal, informative)

### Call-to-Action
- Primary CTA: Brown buttons (Add to Cart, Place Order)
- Secondary CTA: Outlined brown buttons (Continue Shopping)
- Destructive: Red buttons (Remove, Delete)

### Feedback
- Toast notifications for actions
- Loading states for async operations
- Error messages inline and in toasts
- Success confirmations

### Empty States
- Friendly messages
- Clear next actions
- Maintain visual consistency
